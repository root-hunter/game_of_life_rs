## Installation
1. Clone the repo
   ```sh
    git clone https://github.com/root-hunter/game_of_life_rs.git
    ```
2. Move to directory
   ```sh
   cd game_of_life_rs
   ```
3. Install
   ```sh
   sudo make install
   ```
4. Build
   ```sh
   make build
   ```
5. Build
   ```sh
   make serve
   ```
5. Go to <a href="http://localhost:8000">localhost</a>
